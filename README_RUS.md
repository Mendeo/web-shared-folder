# web-shared-folder

## Запуск
```bash
wsf
```
или
```bash
web-shared-folder
```
Доступные параметры:
* --upload or -u - включают режим, позволяющий записывать на сервер.
* [--upload or -u] </path/to/folder/for/sharing> <port> [</path/to/key> </path/to/cert>]
